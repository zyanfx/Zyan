
 This is a project file for the Sandcastle Help File Builder: http://shfb.codeplex.com/

 Howto:

 1. Build the website using SHFB (should take 5-10 minutes), it will create the Help subdirectory.
 2. Fix the casing of subfolders: scripts -> Scripts and styles -> Styles.
 3. Replace ugly icons/online_icon.gif with the provided online_icon.gif
 4. Add missing <br>s to the Zyan project description :)
 5. tar c Help | gzip >help.tgz
 6. Upload to a website, extract to a new folder, enjoy
